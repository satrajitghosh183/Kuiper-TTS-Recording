# TT Otilito Sans Font

To use the [TT Otilito Sans](https://dribbble.com/shots/26425491-TT-Otilito-Sans-Font-Family) font family:

1. Download the font (ensure you have the appropriate license for your use case)
2. Convert TTF/OTF to WOFF2 using [Fontsource converter](https://fontsource.org/tools/converter) or similar
3. Place the following files in this directory:
   - `tt-otilito-sans-regular.woff2` (400)
   - `tt-otilito-sans-medium.woff2` (500)
   - `tt-otilito-sans-semibold.woff2` (600)
   - `tt-otilito-sans-bold.woff2` (700)

Until these files are added, the app uses **Plus Jakarta Sans** from Google Fonts as a similar geometric sans-serif fallback.
